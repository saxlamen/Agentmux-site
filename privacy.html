Agentmux ("we," "our," or "the App") is committed to protecting your privacy. This policy explains how Agentmux handles data security and privacy.

1. No Data Collection

Agentmux is a tool designed for developers. We do not collect, store, or transmit your personal data, server credentials, or terminal traffic to external servers or third parties.

2. Local Storage & Security

  • SSH credentials: Passwords and private keys are stored only in the iOS system Keychain, using Apple-provided security mechanisms.
  • Workspace information: Server configurations, project paths, recent sessions, and recent files are stored locally on your device via SwiftData.
  • Terminal traffic: Communication between Agentmux and your server is encrypted through the SSH protocol. We do not intercept or log terminal output.

3. iCloud Synchronization

If you enable iCloud Sync, supported workspace data such as server settings, recent sessions, and recent files may sync across your own devices through Apple iCloud. We do not have access to that data.

4. Debug Logs

Debug logs are stored locally on your device for troubleshooting. They are not transmitted automatically. You can clear them at any time.

5. Contact

Questions about this policy? Contact us at atmux-support@saxcave.cc.

Agentmux（以下「当社」または「本 App」）は、ユーザーのプライバシーを重視しています。本ポリシーでは、Agentmux におけるデータセキュリティとプライバシーの扱いについて説明します。

1. データ収集について

Agentmux は開発者向けのツールです。当社は、個人データ、server 認証情報、terminal traffic を外部 server や第三者へ収集、保存、送信することはありません。

2. ローカル保存とセキュリティ

  • SSH 認証情報：パスワードと秘密鍵は iOS の Keychain のみに保存され、Apple が提供するセキュリティ機構で保護されます。
  • Workspace 情報：server 設定、project path、recent sessions、recent files は SwiftData を通じてデバイス上に保存されます。
  • Terminal traffic：Agentmux と server の通信は SSH protocol で暗号化されます。当社が terminal output を傍受または記録することはありません。

3. iCloud 同期

iCloud Sync を有効にした場合、server 設定、recent sessions、recent files などの対応する workspace data は Apple iCloud を通じて自身のデバイス間で同期される場合があります。当社はこのデータにアクセスできません。

4. Debug logs

Debug logs はトラブルシューティングのためにデバイス上にのみ保存され、自動送信されません。いつでも削除できます。

5. お問い合わせ

本ポリシーについて質問がある場合は、atmux-support@saxcave.cc までご連絡ください。
